This is a Calendar application for desktops made by JavaFX.

During two weeks, I have worked on establishing the storage unit of the Calendar, enhanced the user interface with the dark mode and parts of the controller. Regarding the storage unit, I used a treeMap to map an event to its date of happening. Add to that, I implemented the Serializable interface to ensure storing the information of its object and keep the Calendar up to date in the next session. For the interface, I have added the "add event" pop up and written a nice styling sheet to make the dark mode.

This was my second experience with agile dev and we have used the MVVC architecture to build it. Although we have worked on it in the days of the pandemic, we were able to use zoom and discord for our meetings, github for composing the code, and trillo to assign tasks for every member of the group. This experience exposed me to a very essential skill for 21st century programers; the ability to work from distance.

This is an adaptation of the famous game Master Mind

In this game the computer (AI) make up a four letter word (from this set{r, o, y, g, b, p}). The player has to attempt to ballpark the AI's guess. The AI supplies the player with the number of letters in correct places and letters in wrong places to help the player.

This game is has only one path of interaction which make it viable for the Model-View-Controller architecture. The model generates the AI's guess and process the user guesses for comparison. The contoller contains the logic of the game and the view is text only view.
